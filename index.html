Muse

Navigate Last.Fm from the command line. Explore similar artist, top songs, etc and stream the video from youtube

Download this project as a .zip file Download this project as a tar.gz file

Usage

$ muse artist name

Here is an example for one of history greatest:

Use the config file to change the number of entries for top tracks and albums.

Install

You'll need jq, youtube-dl and a last.fm api key (see the config file, .muserc)

$ sudo apt-get install youtube-dl jq
$ git clone https://github.com/mihaiolteanu/muse
$ sudo cp muse.sh /usr/local/bin/muse
$ cp .muserc ~/.muserc
