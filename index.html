Usage

$ muse artist name

Use the config file to change the number of entries for top tracks and albums.

Install

You'll need jq, youtube-dl, vlc and a last.fm api key (see the config file, .muserc)

$ sudo apt-get install youtube-dl jq vlc
$ git clone https://github.com/mihaiolteanu/muse
$ sudo cp muse.sh /usr/local/bin/muse
$ cp .muserc ~/.muserc
